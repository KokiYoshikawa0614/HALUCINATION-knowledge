## 設計方針

- プロフィール全体（ヘッダー画像・名前・bio・固定ポスト・リンク）をシステムとして設計
- 名前欄は短くクレデンシャル（AI映画監督）。映画タイトルはbioで数字と一緒に出す
- bio 1行目：クレデンシャル（39分・全編AI・1人）
- bio 2行目：セグメントAの痛みに直接応答（フォロー理由）
- bio 3行目：具体的に何が得られるか
- 京都大学 理学部はbioに入れない（セグメントAのフォロー判断に影響しない）
- トーン：技術的・実証的・簡潔。煽り表現を使わない
- ターゲットの日常語を使う（「設定値」→「設定」、「ワークフロー」→「制作の流れ」）
- 通常ポストにLPリンクは入れない（Xアルゴリズムが外部リンク付き投稿を抑制 + 売り込み感を避ける）
- LP誘導はプロフィールリンク欄と固定ポストに集約
- note記事やYouTube動画のリンクは貼ってOK（自コンテンツへの誘導は自然）

## コンセプトテスト（Gate 2再通過用）

**目標：** 2026-04-30までにセグメントAから10件以上の反応を取得
**通過基準：** コンセプト理解率70%以上 + 反応方向が「教材に興味」寄り

### DM/知人テスト（5人以上目標）

**送信内容：**
```
AI動画のキャラの顔が毎回変わる問題について
発信を始めました。

こういう内容って需要あると思いますか？
率直な感想を聞かせてほしいです。
```
[Before/After画像（pair5_comparison_x.png等）を添付]

```
ちなみにこの技術で作ったのがこれです。
```
https://youtu.be/g3r3_DfQysA

**フレーミングの順序が重要：**
1. 先にBefore/After + コンセプトを提示（教材の文脈を作る）
2. 後から予告編を「証拠」として添える
※逆にすると会話が映画の話になり、コンセプトテストとして機能しない

**判定基準：**

| 返答パターン | 判定 |
|---|---|
| 「需要ある」「自分も困ってた」「教材あったら見たい」 | 理解 + 教材方向 ✅ |
| 「面白いけど自分は困ってない」 | 理解しているがターゲット外 |
| 「映画すごいですね」「作品見たい」 | コンセプト未到達。映画に引っ張られている ❌ |
| 「誰向け？」「何を売るの？」 | コンセプトが伝わっていない ❌ |

### DM送信相手の方針

| 相手タイプ | 優先度 | 扱い |
|---|---|---|
| AI動画生成クリエイター（Kokiがチャンネル登録している数人） | **最優先** | セグメントAに最も近い。この反応をGate 2判定の中心にする |
| WAIFF関連（選ばれた人/選ばれなかったことを公表している人） | 送ってOK | ただしセグメントC寄り。反応は**セグメントAとは分けて集計**する。これだけで10件揃えてもGate 2通過にはならない |

**避けるべき相手：**
- AI動画に関心がない知人（セグメントA外なので反応が無効）
- 大規模アカウント（1万+フォロワー。DMを読まない可能性が高い）
- 映画『HALUCINATION』を既に知っている人（映画寄りにバイアス）
