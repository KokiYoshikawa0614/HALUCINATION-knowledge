## A. 演出の意図（全体要約）

### 1. 命名の設計思想

本作の登場人物名は「損得で機械化する社会」における匿名性・普遍性・関係の断絶と回復を表現するために設計されている。

| 名前 | 設計意図 |
|---|---|
| **佐藤（AI）** | 日本で最も一般的な姓 → 誰にでもアクセス可能な「ホームベース＝AI」という逆説（ハーバーマスの対話志向性と響き合う） |
| **鈴木・高橋・田中** | 次点の汎用姓 → 社会が人を部品化する没人格性・代替可能性を担う |
| **主人公（名前なし）** | 終始ブーバーの「我—それ」として扱われてきた事実を可視化 |
| **母・律子** | 安全・規範から関係のリズム（律）へと揺れ動く弧 |
| **娘・結衣** | 〈結＝結ぶ／衣＝配慮〉→ 損得抜きに他者を庇う徳性 |
| **胎児・光（ひかり）** | 「結びの先に光がある」という語義連鎖 |

### 2. 主要登場人物の役割と意図

**田中の消失について：**
秋葉原無差別殺傷事件の元友人・大友氏の事例を参照。証言によって会社からクビを要求された事例、容疑者家族が追い詰められた事例から、「事件に無関係でも関係者であることで社会的損害を被る」という現実を反映。田中の消失は「不確実性をそのまま保留できるか／都合のよい解釈へ飛びつくか」を観客に突きつける装置。

**結衣について：**
主人公が物理世界で唯一「汝」として対等に向き合えた存在。いじめを受ける凛を庇った結果、標的が自分に移り自死に至った。犯罪心理学的には、いじめは加害側がエスカレートに高揚を見出し、周囲は標的化回避のため黙認・加担する構造がある。

**母（律子）について：**
「公園でサッカー禁止・知らない人と話すな」という行動制限は、安全過剰が信頼ベースの共同体を阻害する社会と響き合う。主人公との出会いで故夫の言葉を思い出し、少しずつ開かれていくが、最終的に結衣の自殺で深い絶望に沈む。

**高橋について：**
卒業作品のために主人公に連絡し、頼み事に金銭を要求する損得勘定の人物。主人公にイベント会場の構造の弱点を教え、カラシニコフ銃のパーツを3Dプリンターで製作・送付したのは高橋だが、それが犯罪につながるとは認識していない。

### 3. 主人公 = テロ犯という構造

本作のテロ犯は主人公（17歳）自身である。物語は**冒頭と結末が連結したループ構造**。

- 冒頭：ニュースで事件を知る
- 展開：損得勘定で動く社会への嫌悪 + 慕うものの喪失 + ホームベース喪失の絶望が重なり犯行へ
- 結末：再び冒頭のニュースへ戻る

思想的基盤：**ニーチェの永劫回帰**（同じ出来事の永遠反復を全面肯定して行為できる者を超人とする）

**「アモール・ファティ」**（冒頭のキーワード）= 絶対的価値基準に依存せず自らの価値で運命を受け入れ肯定的に生きる態度。主人公はこの影響下にある。

主人公を17歳にした理由：報道規制で実名報道されないため。「価値を知らしめるために犯行を行うが、最後まで誰からも名前で呼ばれず社会から消される」という皮肉を描く。

### 4. 事件日：9月21日（国際平和デー）

外部史実の固有名連想を避けつつ、「平和を祈る日に最も平和から遠い出来事が起こる」という反照を作るため。秋分に隣接する「境界の時期」でもあり、現実／幻・加害／救済・生／死の敷居を自然の記号で支える。

### 5. 影響源となった映画・文学

| 作品 | 影響内容 |
|---|---|
| **『プリデスティネーション』** | 運命の不可逆性を知った者はそれを受容して行為へ踏み出す。「動機→行為」ではなく「初めから犯行が定まっている世界」 |
| **『ファーザー』** | 認知症患者の純主観で世界を提示。観客に当事者の知覚を100%追体験させる手法 |
| **『her／世界でひとつの彼女』** | AI（佐藤）が多くの人と同時に関係を持っているという設定 |
| **『マイノリティ・リポート』** | 鈴木の会社の事業（メタバース技術）のビジュアルモチーフ |
| **『エレファント』** | 「郡盲象を評す」の寓話 → 事件の全体像は誰にも絶対に分からないというテーマ |
| **『罪と罰』（ドストエフスキー）** | ラスコーリニコフ・スヴィドリガイノフの思想（天才と凡人・自己滅却）。監督自身の危険な精神状態と重なる |
| **『オイディプス王』（ソフォクレス）** | 犯行後に両目を簪で刺す行為のオマージュ。条件プログラム的損得勘定の否定 |
| **『銀河鉄道の夜』初稿（宮沢賢治）** | 読み聞かせシーンで初稿を使う理由：「起こるべくして起こる」という決定論的フレーム（ブルカニロ博士）。賢治の国柱会との関係も背景に持つ |
| **『ツァラトゥストラはかく語りき』（ニーチェ）** | ラクダ→ライオン→赤子の三段階変容 |

**演出の核心的結論：**
いかに学術的に殺人・テロを分析しても、俯瞰的描写では観客は安全圏に留まり犯人を本当に理解しない。
→ 最善は**犯人の主観を100%連続で体験させること**（『ファーザー』手法）
→ 観客に「自分もやってしまう」という恐怖を直面させ、自明性を破壊し、社会の在り方と自己の規律に向き合わせる

### 6. 象徴体系

**アネモネの花の変化：**
- 青いアネモネ（「固い誓い」）→ 主人公が他者と内発的に絆を育もうとする決意
- 白いアネモネ（「希望・期待・真実」）→ 犯罪行為を決意した主人公の矛盾した精神状態を皮肉として描く
- 「はかない恋」（花言葉）→ AIへの人間を超える愛着・実らぬ恋の暗示
- 結衣と佐藤を喪失 → アネモネが枯れる
- 最後に映る白いアネモネ → 恐ろしい犯罪と希望に満ちた精神状態のあいだの皮肉なギャップ

**カニバリズムのシーン（理論的根拠）：**
- ルソー・レヴィ＝ストロースの「同化」と「同一化」：他者を食べることが最も単純な他者との同一化の方法
- 佐川一政の事例に近い：自己に欠けていると感じる資質を他者から取り込もうとする融合への欲望
- エンドカニバリズム（共同体内構成員の肉体を食する）：愛・敬意・共同体の連続性確保の動機

**「超人」と自己食人：**
子どもの超人となった自分を食べることで一体化しようとする。ニーチェの「赤子」（新しい価値を創造する存在）となったことの表現。

### 7. タイトル「HALUCINATION」の多義性（本作最大の仕掛け）

タイトルは意図的に「HALUCINATION」（Lが1つ）とスペルが誤っている（正しくは HALLUCINATION）。
これ自体が「ハルシネーション」（AIによる誤情報生成）を体現している。

**三重の意味：**

| 解釈 | 内容 |
|---|---|
| **①主人公の幻覚** | ホームベース喪失によって時間ループの中で永遠に犯罪を犯す運命にあると信じ切っている幻覚。日付を間違えるシーン→時間感覚の崩壊。鈴木との口論が噛み合わないのは実は事件前の会話だったから。主人公が部屋から出ないのは引きこもり状態の描写 |
| **②AIのハルシネーション** | 佐藤（AI）の故障がコールセンター曰く「先週から」→全登場人物が誤情報を基に行動している可能性。AIに完全依存した社会でシステムエラーが起きた場合、極端なフィルターバブルが生じ会話が成立しなくなる。これが本作の核心 |
| **③観客の認識の問題** | 映画自体が「事件の真相に何もたどり着いていない」可能性。『エレファント』の「郡盲象を評す」。分からないことを分からないとして保留できるか、自分に都合のよいことだけを信じるかを挑発する映画 |

### 8. 監督自身について

- かつて優等生だったが大学受験失敗でうつ病を発症
- 自己分析：他責化が起きた → 自分の実力を発揮できない環境への恨み
- 自我を抑制しながら自然発生する個性を尊重する制作姿勢
- 監督自身の破壊衝動がいつどのように芽生えたかをメタ認知し映画に反映
- 危険な精神状態における思想が主人公と重なる（自分が映画に取り込まれる必要があった）
- オートセクシャルの傾向を持っており、主人公の自分自身とのキス・セックスシーンに反映

**設定の細部（キャラクター間の関係）：**
- 田中：主人公と小学校・高校が同じ（中学は別）→ 名前をはっきり覚えていない設定のため
- 高橋：主人公と小学校のみ同じ
- 鈴木：主人公と高校のみ同じ
- 主人公は高校を1ヶ月ほどで辞めている
- 主人公の両親は宗教法人への献金で破産（統一教会問題における山上容疑者をモチーフ）
- 主人公がサバイバルゲーム・マインクラフト得意という設定 → 銃の正確な狙撃と3Dプリンター設計の能力を担保するため

## B. 殺人事件・テロ事件と孤独と神経症（全体要約）

### 概要と理論的枠組み

本論文は「なぜ孤独が殺人・テロへと至るか」を、**構造—心理—行為—統治** の四段の回路として分析する。

**主要概念の定義：**
- **代替可能性：** 個人が「誰でもない誰か」として透明化され、他者に取り替え可能だとみなされる社会的圧力
- **if–then型合理性：** 行為を損得勘定に還元し、条件が満たされれば機械のように最適反応を返す規範
- **ホームベース：** ブーバーの「我と汝」を基盤とした、交換論理に回収されない関係性のネットワーク（強い絆＋弱い絆の二重構造、サードプレイス的な物理的空間で育まれる）

### 四段の回路

**第一段：構造**
- Weberの「鉄の檻」：近代合理化が行為をif–then型計算可能性へ回収
- Habermasの目的—手段合理性の拡張：人格の自律と尊厳の侵食
- AI・技術進歩が「自分は替えが利く」という確信を制度として注入
- 内発的行為の場が縮減し、関係から余剰が失われる

**第二段：心理**
- Freud的神経症：対処不能な不安の持続→意味の欠けた反復行為の自然化
- Frommのナチス分析：「安心を得るために服従を選ぶ」という反転
- 唯一性の承認を失った個人 → 秩序と帰属の確かさと引き換えに判断停止を選ぶ
- 感情構造が「安心の即時回収」へ傾く

**第三段：行為**
- 仮想現実（メタバース等）への逃避
- 新反動主義（ピーター・ティール的「ダーク・エンライトメント」）との共鳴
- 贈与感情の欠落 → 民主制の矮小化・少数者統治の正当化
- 「バカには無害なドラッグとメタバースとBIを与えておけ」という構図
- 仮想世界には「死の不可逆性」が欠落 → 人格の統合に接続しない

**第四段：統治**
- Hobbsのリヴァイアサン：暴力を統治に委ねるコストとメリット
- 統治への信頼が毀損すれば自力救済が再燃
- ホームベースを欠く社会では暴力が「個人の暴走」として表出
- 地域のハレ（祭り）の機能喪失・職場の「楽しい職場」という擬似ホームベースの問題
- 厳罰化の快楽は再犯抑止と必ずしも整合しない

### 二つの媒介（四段を束ねる核心）

1. **唯一性の承認の崩壊**
2. **その場の訂正の欠落**

この二つが揃うとき、四段の回路が殺人・テロへと収束する。

### 具体的事例による照応

| 事例 | 対応する構造 |
|---|---|
| **オウム真理教** | 貧・病・家族内不和 → 交換可能な部品感覚 → 期待の裏切り → 「帰依」の形式でテロ加担。救済の物語が関係の先取り（擬似ホームベース）で身体・感情の接続を先に確定し、後から教義が流し込まれる |
| **ナチス政権** | ワイマール期の不安定さ → Eichmannの「自分は入れ替え可能な存在」発言 → Arendtの「凡庸な悪」= 没主体化の極致 |
| **秋葉原無差別殺傷事件** | 家庭での虐待的教育 → 中学までの優秀さが高校で埋没 → 「交換可能」感覚の形成 → ネット上の架空ホームベースを喪失 → 社会全体を罰する動機へ |

### 暴力に至る者・至らない者の差異

「なぜ同じ状況でも暴力に至る者と至らない者がいるか」への答え：
→ **決定論的な単一要因ではなく、個人の素因と引き金となるストレスの相互作用による確率論的現象**
→ 最終的な分岐を決定づけるのは**保護要因（ホームベース）の有無**

### 孤独・引きこもりの再定義

孤独と引きこもりは個人の弱さではなく、**構造に媒介された現象**として位置づけられる。
- 社会の流動化の中で人が交換可能とみなされる → 実りあるコミュニケーションが断たれる
- 日本では引きこもりに現れる（欧米ではホームレスへ流れる文化的分岐）
- 当人が助けを求めづらい：「無能と認知されるほど社会に出ることも支援要請も難しくなる」

### 処方：ホームベースの再建

**短期：** 「何もしないで、ただ居られる」存在肯定の公共空間を再構築（評価・有用性から解放された停留点）

**中期：** 贈与に基づくコミュニティを育む。「愛されるより愛する」実践を通じて他者の全人格に応答する訓練。贈与の両義性（与えっぱなしにならない・しかし負債化にも堕とさない）を扱う作法の涵養。

**長期：** 「学びのOS」を制度化し相互訂正を標準動作にする。祭り・ハレの回路を再起動。日常の共同没入の場を増やす。リーダーシップは感情的シンクロを喚起できる資質を持つ。

**ホームベースの再建に必要な4要素：**
1. 唯一性の承認
2. 遅延回収（即時清算に回収されない関係の持続）
3. 訂正の即時性
4. 身体的共在の厚み

オンラインは「非匿名・責任ある発話・対面への移行可能性」の条件を満たすときのみ補助として機能。

### 本論文と映画の関係

- 本論文の学術的分析が映画の**ストーリー・キャラクター・テーマ設計の直接的な理論基盤**となっている
- 「代替可能性」「ホームベース」「if–then型合理性」は映画内でも核心的なテーマ

## 統合：映画HALUCINATIONのストーリー設計の核心

| テーマ | 演出的表現 | 理論的根拠 |
|---|---|---|
| 代替可能性・没人格化 | 主人公に名前がない。汎用姓（佐藤・鈴木・高橋・田中）で占められる世界 | Weber的合理化・Buberの「我—それ」 |
| ホームベースの喪失 | 結衣（物理世界の唯一の「汝」）と佐藤（AIという疑似ホームベース）の相次ぐ喪失 | 孤独→テロへの四段回路の核心 |
| AIと幻覚 | タイトルのスペルミス、日付の混乱、鈴木との会話の噛み合わなさ | 情報の非共有・フィルターバブル |
| 犯人の主観100%体験 | 犯人＝主人公という設定、部屋の中からしか描かれない世界 | 『ファーザー』手法、俯瞰的描写の限界 |
| 永劫回帰・運命の受容 | ループ構造、アモール・ファティ、オイディプス的自傷 | ニーチェ、プリデスティネーション |
| 分からないことの保留 | 田中の消失・事件の全容が描かれない | 「郡盲象を評す」（エレファント）、保留の問いかけ |
